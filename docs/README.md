# Documentation

If you are new to the project, read `../README.md` first, then use this index for details.

## I want to...

- Encrypt/decrypt quickly and learn the CLI: [Usage and CLI](USAGE.md)
- Understand key lookup and rotation: [Key Management](KEY_MANAGEMENT.md)
- Share encrypted files with multiple recipients: [Asymmetric Mode](ASYMMETRIC.md)
- Set up CI/CD, Docker, or Kubernetes: [Deployment](DEPLOYMENT.md)
- Understand the on-disk encrypted format: [File Format](FILE_FORMAT.md)
- Install editor support: [Editor Integration](EDITORS.md)

## Full guide list

- [Usage and CLI](USAGE.md): install, quickstart, runtime loading, command overview.
- [Key Management](KEY_MANAGEMENT.md): key resolution order, rotation, and handling practices.
- [Asymmetric Mode](ASYMMETRIC.md): multi-recipient workflow and revocation caveats.
- [Deployment](DEPLOYMENT.md): GitHub Actions, Docker, and Kubernetes examples.
- [File Format](FILE_FORMAT.md): `v=1` / `v=2` encrypted `.env` layout and metadata.
- [Editor Integration](EDITORS.md): installing the VS Code / Cursor extension.
